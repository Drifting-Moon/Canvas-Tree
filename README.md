# Canvas 🎨

A minimalist, interactive sticky-note application built with React and TypeScript.

## 🚀 Features
- **Click to Add**: Add new notes anywhere on the canvas.
- **Drag & Move**: Reposition notes freely.
- **Resize**: Adjust the size of each note.
- **Auto-Save**: Notes are saved to your browser's local storage.
- **Dynamic Colors**: Every new note gets a unique color.

## 🛠️ Tech Stack
- **React 18**
- **TypeScript**
- **Vite**
